# Пакет Or

Пакет, предоставляющий функцию для объединения нескольких каналов *done*.

```go
merged := or.Or(sig(100*time.Millisecond), sig(200*time.Millisecond))
<-merged // разблокируется, когда первый канал отправит сигнал
```

Пакет также включает подробные тесты, демонстрирующие поведение при нуле, одном и множестве каналов.
